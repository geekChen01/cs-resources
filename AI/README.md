### 课程实验（Berkeley CS188 Pacman）

基于 UC Berkeley CS188 课程实验框架，使用 Pacman 游戏作为教学平台。

- **实验一：搜索算法**（`课程实验/search/`）
  - 实现 DFS、BFS、UCS、A* 等搜索算法
  - 核心：`search.py`、`searchAgents.py`
  - 运行：`python pacman.py`

- **实验二：多智能体**（`课程实验/multiagent/`）
  - 实现 Minimax、Alpha-Beta 剪枝、Expectimax
  - 设计评估函数
  - 核心：`multiAgents.py`
  - 运行：`python pacman.py -p MinimaxAgent`

- **实验参考代码**（`课程实验/pacman/`）
  - `1/`：搜索实验参考实现
  - `2/`：多智能体实验参考实现
  - `1.docx`、`2.docx`：实验报告模板
